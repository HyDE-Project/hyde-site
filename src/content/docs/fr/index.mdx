## Qu'est‑ce que HyDE ?

HyDE est un environnement de développement esthétique, modulaire et conforme XDG
pour Hyprland sur Arch, axé sur un design propre, la maintenabilité et une UX soignée.
Les scripts cœur sont séparés de votre configuration pour des mises à jour plus sûres.

## Installer

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Besoin d'aide ? Voir : [Premiers pas > Installation](/fr/getting-started/installation)
