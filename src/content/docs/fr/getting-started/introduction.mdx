HyDE est un environnement de développement propre, esthétique et modulaire pour
Hyprland sur Arch. Il met l'accent sur une expérience cohérente, des
paramètres par défaut pertinents et des mises à jour qui ne gênent pas.

## Ce que vous trouverez ici

- Un guide d'installation étape par étape et des conseils pour le premier démarrage
- Une organisation modulaire et opiniâtre pour garder une configuration propre
- Des outils de theming et une galerie communautaire pour partager des styles
- Des références pour les utilitaires et commandes fournis

## Installation rapide

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Après l'installation, redémarrez, puis connectez‑vous à HyDE depuis le display manager.
