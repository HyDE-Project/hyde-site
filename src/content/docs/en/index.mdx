## What is HyDE?

HyDE is an aesthetic, modular, and XDG-compliant development environment for
Hyprland on Arch, focused on clean design, maintainability, and delightful UX.
It separates core scripts from user config, making updates safer and easier.

## Install

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Need help? See the full guide: [Getting Started > Installation](/en/getting-started/installation)
