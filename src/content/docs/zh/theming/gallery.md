## 使用方法

> - 确保你已安装 [HyDE](https://github.com/HyDE-project/hyde)

```shell

> + 在终端中执行 "theme.import.py"

> + 选择你喜欢的主题
```

![图片](https://raw.githubusercontent.com/HyDE-Project/hyde-gallery/43e6cce54698f24842ae2fbb7b80e79556a21e02/assets/HydeThemes.png)

---

:::note

HyDE 目前尚未使用命令行界面(CLI)，因此我们需要手动调用脚本。请注意，未来的 CLI 仍将调用 `theme.import.py` 和 `theme.patch.sh`，因此不会有明显的 API 变化。

:::

📢 - 你也可以添加带单引号的 URL 链接或文件路径来安装特定主题

```shell
theme.patch.sh "主题名称" "/配置文件路径"
theme.patch.sh "主题名称" "https://github.com/用户/仓库"
theme.patch.sh "主题名称" "https://github.com/用户/仓库/tree/分支"
```

示例

```shell
theme.patch.sh 'Hack the Box' 'https://github.com/HyDE-Project/Hyde-gallery'
```

:::tip
📢 如果主题有归档更新，设置环境变量 `FORCE_THEME_UPDATE=true` 来更新 Gtk、图标、光标和字体。

```shell
 export FORCE_THEME_UPDATE=true
```

:::

### 更新

你可以使用 `theme.import.py` 脚本来获取/更新本地主题。

<details>
<summary>获取所有主题</summary>
 这个命令会更新位于 `xdg_config/hyde/themes/` 中列在 [HyDe-gallery](#主题库) 的所有主题

```bash
theme.import.py --fetch all
```

</details>

<details>
<summary>获取特定主题</summary>
 这个命令会更新位于 `xdg_config/hyde/themes/` 中列在 [HyDe-gallery](#主题库) 的特定主题

```bash
theme.import.py --fetch "主题名称"
```

</details>
