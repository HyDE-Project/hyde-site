## 什么是 HyDE？

HyDE 是一个面向 Arch 上 Hyprland 的、美观且模块化并符合 XDG 的开发环境，
专注于干净的设计、可维护性以及良好的用户体验。它将核心脚本与用户配置分离，
让升级更安全、冲突更少。

## 安装

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

需要帮助？请参阅：[入门 > 安装](/zh/getting-started/installation)
