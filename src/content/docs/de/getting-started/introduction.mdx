HyDE ist eine saubere, ästhetische und modulare Entwicklungsumgebung für
Hyprland auf Arch. Der Fokus liegt auf einem konsistenten Look & Feel,
sinnvollen Defaults und Updates, die nicht im Weg stehen.

## Was Sie hier finden

- Geführte Installation und Tipps für den ersten Start
- Meinungsstarke, modulare Struktur für aufgeräumte Konfigurationen
- Tools fürs Theming und eine Community‑Galerie zum Teilen von Styles
- Referenzen zu mitgelieferten Utilities und Kommandos

## Schnelle Installation

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Nach der Installation neu starten, im Display‑Manager HyDE auswählen und anmelden.
