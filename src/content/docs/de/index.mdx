## Was ist HyDE?

HyDE ist eine ästhetische, modulare und XDG‑konforme Entwicklungsumgebung für
Hyprland auf Arch – mit Fokus auf sauberes Design, Wartbarkeit und gute UX.
Kernskripte sind von der Nutzerkonfiguration getrennt, wodurch Updates sicherer
und konfliktfrei werden.

## Installation

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Hilfe nötig? Siehe: [Erste Schritte > Installation](/de/getting-started/installation)
