## ¿Qué es HyDE?

HyDE es un entorno de desarrollo estético, modular y compatible con XDG para
Hyprland en Arch, centrado en un diseño limpio, mantenibilidad y una UX agradable.
Separa los scripts core de tu configuración, facilitando actualizaciones seguras.

## Instalar

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

¿Necesitas ayuda? Consulta: [Primeros pasos > Instalación](/es/getting-started/installation)
