HyDE es un entorno de desarrollo limpio, estético y modular para
Hyprland en Arch. Se enfoca en una apariencia consistente, valores
predeterminados sensatos y actualizaciones que no estorban.

## Qué encontrarás aquí

- Flujo de instalación guiado y consejos para el primer inicio
- Estructura modular y opinada para mantener la configuración ordenada
- Herramientas de theming y una galería comunitaria para compartir estilos
- Referencias de utilidades y comandos incluidos

## Instalación rápida

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Tras la instalación, reinicia, entra al display manager y accede a HyDE.
