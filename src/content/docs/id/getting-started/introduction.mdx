HyDE adalah lingkungan pengembangan yang bersih, estetis, dan modular untuk Hyprland di Arch Linux.
Fokusnya adalah pada tampilan dan nuansa yang konsisten, konfigurasi bawaan yang masuk akal, 
serta pembaruan yang tidak mengganggu alur kerja Anda.
## Apa yang dapat Anda temukan di sini

- Alur instalasi yang disertai tips untuk boot pertama
- Tata letak modular yang teratur untuk menjaga konfigurasi bersih
- Alat tema dan galeri komunitas untuk berbagi gaya
- Dokumentasi referensi untuk utilitas dan perintah terintegrasi

## Instalasi Cepat

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Setelah instalasi selesai, reboot ke display manager dan login ke HyDE.
