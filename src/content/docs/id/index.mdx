## Apa itu HyDE?

HyDE adalah lingkungan pengembangan yang bersih, estetis, dan modular untuk Hyprland di Arch Linux.
Fokusnya adalah pada tampilan dan nuansa yang konsisten, konfigurasi bawaan yang masuk akal, 
serta pembaruan yang tidak mengganggu alur kerja Anda.

## Instalasi

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Butuh bantuan? Lihat panduan lengkap: [Perkenalan > Instalasi](/id/getting-started/installation)
