---

**SwayOSD** को **HyDE** द्वारा पहले से ही सपोर्ट किया जाता है। इसे काम में लाने के लिए आपको केवल डेमन (daemon) को मैन्युअली जोड़ना होगा।

### त्वरित सेटअप

```bash
  exec-once = swayosd-server # swayosd सेवा को सक्षम करें
  exec-once = $(pkexec swayosd-libinput-backend) # कीबोर्ड इनपुट के लिए swayosd सेवा (एक सबशेल में चलाना आवश्यक है)
```
