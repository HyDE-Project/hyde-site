## HyDE क्या है?

HyDE एक सौंदर्यपूर्ण, मॉड्यूलर और XDG‑अनुरूप विकास वातावरण है
जो Arch पर Hyprland के लिए बनाया गया है — साफ डिज़ाइन, रखरखाव‑योग्यता
और आनंददायक UX पर केंद्रित। यह कोर स्क्रिप्ट्स को यूज़र कॉन्फ़िग से अलग
रखता है ताकि अपडेट्स सुरक्षित और आसान हों।

## इंस्टॉल

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

मदद चाहिए? देखें: [शुरुआत > इंस्टॉलेशन](/hi/getting-started/installation)
