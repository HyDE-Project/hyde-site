## त्वरित इंस्टॉल

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

इंस्टॉल के बाद रीबूट करें और डिस्प्ले मैनेजर से HyDE में साइन‑इन करें।
