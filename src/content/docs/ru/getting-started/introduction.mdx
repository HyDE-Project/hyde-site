HyDE — это чистая, эстетичная и модульная среда разработки для
Hyprland на Arch. Акцент на единообразии интерфейса, разумных
настройках по умолчанию и обновлениях, которые не мешают работе.

## Что вы найдёте

- Пошаговую установку и советы для первого запуска
- Продуманную модульную структуру для аккуратных конфигураций
- Инструменты оформления и галерею сообщества для обмена стилями
- Справочник по утилитам и командам из поставки

## Быстрая установка

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

После установки перезагрузитесь и войдите в HyDE через дисплей‑менеджер.
