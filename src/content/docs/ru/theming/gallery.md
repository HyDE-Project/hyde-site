## Использование

> - Убедитесь, что у вас установлен [HyDE](https://github.com/HyDE-project/hyde)

### Интерактивный импорт тем из галереи

```shell
hydectl theme import
```

### Импорт определенной темы по названию и URL-адресу

```shell
hydectl theme import --name "Название темы" --url "https://github.com/User/Repository"
```

### Обновление всех локальных тем

```shell
hydectl theme import --fetch all
```

### Обновление определенной темы

```shell
hydectl theme import --fetch "Название темы"
```

### Интерактивный выбор темы

```shell
hydectl theme select
```

### Установка темы напрямую

```shell
hydectl theme set "Название темы"
```

## Управление обоями

### Интерактивный выбор обоев

```shell
hydectl wallpaper select
```

### Установка определенных обоев

```shell
hydectl wallpaper set /path/to/wallpaper.jpg
```

### Список доступных обоев

```shell
hydectl wallpaper list
```
