## Что такое HyDE?

HyDE — эстетичная, модульная и XDG‑совместимая среда разработки для Hyprland на
Arch. Акцент на чистом дизайне, удобстве сопровождения и приятной UX.
Основные скрипты отделены от пользовательских конфигов для безопасных обновлений.

## Установка

```sh
pacman -S --needed git base-devel
git clone --depth 1 https://github.com/HyDE-Project/HyDE ~/HyDE
cd ~/HyDE/Scripts
./install.sh
```

Нужна помощь? См.: [Начало работы > Установка](/ru/getting-started/installation)
